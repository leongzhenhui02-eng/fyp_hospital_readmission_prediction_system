# fyp_hospital_readmission_prediction_system

1.	Performed data preprocessing and cleaning on healthcare datasets, including handling missing values, encoding categorical variables, and feature selection
2.	Conducted exploratory data analysis (EDA) to identify key factors influencing hospital readmission, such as patient demographics and medical history
3.	Engineered meaningful features to improve model performance and enhance predictive accuracy
4.	Developed and evaluated multiple machine learning models (e.g., CatBoost, Random Forest) for readmission prediction
5.	Applied Explainable AI (XAI) techniques (SHAP) to interpret model predictions and identify critical risk factors
6.	Built visualizations to communicate insights on patient risk profiles and readmission patterns
7.	Achieved accurate prediction performance and provided data-driven recommendations to support healthcare decision-making
